## üåê Making Repository Public

### Step 1: GitHub Repository Settings
1. Go to: https://github.com/bassslap/netbox-wug-sync/settings
2. Scroll down to "Danger Zone"
3. Click "Change repository visibility"
4. Select "Public"
5. Confirm by typing the repository name

### Step 2: Enable GitHub Actions (if not already enabled)
1. Go to "Actions" tab in your repository
2. Enable GitHub Actions if prompted
3. The CI/CD pipeline will automatically run on future pushes/PRs

### Step 3: Create GitHub Release
1. Go to: https://github.com/bassslap/netbox-wug-sync/releases
2. Click "Create a new release"
3. Choose tag: `v0.1.0`
4. Release title: `NetBox WUG Sync v0.1.0 - Initial Public Release`
5. Copy the description from the tag annotation
6. Attach any additional assets if needed
7. Check "Set as the latest release"
8. Click "Publish release"
